Hypertext Markup Language (HTML)

HTML merupakan bahasa markup standar untuk membuat halaman web. HTML mendeskripsikan struktur dari halaman web menggunakan markup. Elemen-elemen HTML adalah blok-blok yang membangun halaman HTML.Elemen-elemen HTML direpresentasikan melalui tags.

Web Browser tidak menampilkan tag HTML, tetapi menggunakannya untuk me-render konten halaman web. Bahasa markup bukan bahasa pemrograman. Oleh karena itu, penyebutan bahasa pemrograman HTML adalah kurang tepat. Editor yang dapat digunakan untuk membuat halaman HTML antara lain: Sublime Text, Notepad++,VS Code

Struktur Dokumen HTML
  • HTML
    • Dokumen HTML harus diawali dengan tag < html > dan diakhiri dengan < /html >
    • Digunakan untuk memberi tahu kepada web browser bahwa yang berada dalam kedua tag tersebut adalah dokumen HTML
  • HEAD
    • Bagian header dalam dokumen HTML
    • Biasanya berisi judul halaman web
    • Judul halaman Web akan muncul pada tab web browser
  • BODY
    • Bagian isi dari dokumen HTML
    • Dapat disisipkan teks, gambar, video maupun link
    • Isi yang ada di dalam body akan ditampilkan pada halaman web
    • Tag Body dapat pula ditambahkan atribut bgcolor untuk pewarnaan background dan warna tulisan dengan text diisi nilai warna.

Cascading Style Sheet (CSS)

CSS = Cascading Style Sheet adalah teknik penulisan kode untuk memperindah dan mempermudah dalam pengkodean HTML dengan tujuan untuk memperindah tampilan website.

CSS dimulai dengan : < STYLE TYPE="text/css">NamaKodeBaru { Parameter : value }< /STYLE>

Sebagai contoh untuk kode HTML < B>..< /B> artinya adalah cetak tebal pada teks yang terletak diantaranya. Dengan CSS, dapat mendefinisikan < B>..< /B> agar memiliki efek tambahan sesuai yang diinginkan, misalnya efek terhadap warna huruf.

ID dan Class Selector
  • ID
    • Id selector menggunakan atribut id untuk memilih elemen tertentu.
    • id bersifat unik di dalam satu halaman, id selector digunakan untuk memilih satu elemen unik (tidak boleh ada id yang sama dalam satu halaman)
    • Sintak CSS untuk memilih elemen berdasarkan id selector: #nama_id (karakter pagar diikuti nama id dari elemen terkait).
  • CLASS SELECTOR
    • Class selector menggunakan atribut class untuk memilih elemen tertentu.
    • Sintak CSS untuk memilih elemen berdasarkan class selector .nama_kelas (titik diikuti nama kelas)
